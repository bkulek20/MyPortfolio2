How to Host A Website from Github?

Proje Yapısını Oluşturmak

İlk adım olarak GitHub’da MyWebsite2 adında public bir repository oluşturdum ve projeyi bilgisayarıma şu komutla klonladım: git clone "clone_url"

GitHub repo linkime buradan ulaşabilirsiniz: https://github.com/bkulek20/MyWebsite2

Daha sonra çok basit bir index.html dosyası oluşturdum.

GitHub Pages kullanarak projeyi yayına almak için şu adımları izledim:

• GitHub'da Settings → Pages sekmesine gittim.
• Build and Deployment kısmından Source seçeneğini Deploy from a branch olarak ayarladım.
• main branch’imi seçip kaydettim.

Kısa bir süre sonra site otomatik olarak yayınlandı ve bağlantı üzerinden erişilebilir hale geldi.

After that, I planned my web app’s structure as followed:

index.html -> for the frontend
main.css -> for styling
app.js -> for adding functionality
I used HTML, CSS, and JavaScript for development.

To commit a change from VS Code to GitHub, I used the following commands:

git add .
git commit -m “comment”
git push origin main

To access my website and host it on GitHub, I first filled the index.html and main.css files. This displayed the content of my website.

To host the website from Github, I went to Settings -> Pages -> selected my main branch -> Save. Then, the link of my website became visible.

I realized that changes that I committed to GitHub were reflected on my website with a slight delay.
